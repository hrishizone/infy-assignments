
# Assignment 1: AWS S3-Lambda File Copy Automation

This assignment demonstrates an AWS automation workflow where an **S3 bucket triggers a Lambda function** automatically when a new object is uploaded, and the Lambda **copies the newly added object to a destination bucket**.

## Technologies Used
- **AWS S3** 
- **AWS Lambda**
- **AWS CloudFormation** 
- **AWS IAM**
- **Python (Boto3)**
